# Emotion Detector Web Application

## 📌 Project Overview
This project is an AI-based Emotion Detector web application developed using IBM Watson Natural Language Understanding (NLU).

The application analyzes user input text and detects emotions such as:
- Joy
- Anger
- Disgust
- Fear
- Sadness

It also determines the dominant emotion.

---

## 🚀 Features
- Emotion analysis using Watson NLP
- Flask-based web interface
- REST API endpoint
- Error handling
- Unit testing
- Static code analysis support

---

## 🛠️ Technologies Used
- Python
- Flask
- IBM Watson NLU
- HTML / JavaScript
- Requests library
